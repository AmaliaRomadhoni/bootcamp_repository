## About Laravel Starter Template

This is a Laravel Starter Template for your first projects, included Laravel/ui with vue js Scaffholding and AdminLTE Template

## authtentication

-- Have a simple role management included ['admin', 'user']

## How to Use

1. Clone this repository to your project
2. Run `composer update`
3. Generate the application key using `php artisan key:generate`
4. Run database migrations and seed the database using `php artisan migrate --seed`
5. Now you have a Laravel project with the Starter Template
